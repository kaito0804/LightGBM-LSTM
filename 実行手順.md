#### **vps上で**
# 1. フォルダに移動
cd /opt/botter/LightGBM_LSTM/

# 2. 仮想環境を起動 (左側に (venv) と出るはずです)
source venv/bin/activate

# 3. 再度起動
nohup python -u main.py > bot.log 2>&1 &

# 4. 数秒待ってから確認
ps aux | grep main.py

# 再学習
rm models/lgb_ETH.pkl
rm models/lstm_ETH.h5
python train_models.py


**本番稼働（バックグラウンド実行）**
pkill -f main.py
nohup python -u main.py > bot.log 2>&1 &



**ボットが今何をしているか見る**
tail -n 100 bot.log
tail -f bot.log （リアルタイムログ）
※ Ctrl + C でこの表示を閉じても、ボット自体は止まりませんので安心してください。



**ボットの動きを止めたいとき**
pkill -f main.py



**ログファイルを空にする（ファイルが大きくなりすぎた時）**
> bot.log

**main.pyの最終更新日時確認**
ls -l main.py
ls -l .env
